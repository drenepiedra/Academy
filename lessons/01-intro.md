# Bienvenido al Curso

Como inicio este curso te enseñará:

- Fundamentos de HTML
- Fundamentos de CSS
- Fundamentos de JavaScript
- Fundamentos de SQL

## ¿Qué es este curso?

Este curso es una guía para aprender a desarrollar sitios web desde cero. Nos enfocaremos en los fundamentos de HTML, CSS y JavaScript, pero también cubriremos conceptos básicos de SQL.

## ¿Cómo puedo empezar?

Si estás interesado en aprender a desarrollar sitios web, te recomendamos que comiences con los fundamentos de HTML y CSS. Estos conceptos son fundamentales para entender cómo funcionan los navegadores web y cómo se muestran las páginas web.

Para aprender a desarrollar sitios web, te recomendamos que comiences con JavaScript. Este es el lenguaje más popular para desarrollar sitios web y es muy versátil. Además, JavaScript es un lenguaje de programación que se utiliza en muchos frameworks web modernos.

¡Hgámoslo!
